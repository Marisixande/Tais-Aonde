Tais Aonde, seria um site para auxiliar quem visita a cidade de Florianópolis-SC e não sabe nem se localizar, nem se as avaliações dos demais sites são reais. Seria possível ver avaliações de lugares de acordo com seu objetivo com a viagem, criar rotas, utilizar um sistema de GPS, encontrar hospedagens e (caso a ideia seja levada para frente) ganhar descontos em certos estabelecimentos. De inicio o usuario apenas teria acesso a 5 avaliações e o sistema GPS e o resto das funcionalidades so ficariam disponiveis para aqueles que pagam um plano mensal, caso se interesse nas outras ofertas. Também existiria a opção por um plano especifico para locais da área conseguirem anunciar seus servisos de estadia.
